The MMI Ontology Registry and Repository (ORR) is a web application through which you can create, update, access, and map ontologies and their terms.

Background information about the ORR, including links to related information, is provided at the MMI Semantic Web Services and Tools page. More information about the semantic framework, of which ORR forms a part, can be found at the MMI Semantic Framework page.
